# Production Ops

This section is for production hardening and runtime operations.

- Start with [factory](./factory)
- Configure plugins with [plugins](./plugins)
- Use [events](./events) for event transport choices (memory, Redis Pub/Sub, Redis Streams) and SSE broadcasting
- Use [workflows](./workflows) for in-process compensation (`withCompensation`) and durable workflows (Streamline integration)
- Use [idempotency](./idempotency) for exactly-once request semantics
- Use [audit](./audit) for change tracking strategy
- Debug multi-tenant issues with [tenant-pipeline](./tenant-pipeline)
- Export docs via [openapi](./openapi)
