## Key Features

- **Auth**: Discriminated union — JWT, Better Auth, custom plugin, custom function, or disabled
- **QueryCache**: TanStack Query-inspired server cache with SWR and auto-invalidation on mutations
- **Runtime modes**: `memory` (zero config) or `distributed` (Redis stores)
- Arc is database-agnostic. MongoKit is one adapter, not a requirement.
